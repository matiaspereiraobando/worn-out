# Style / Mood Board

Drop Midjourney (or any) style references here so they can be used to define the
game's palette and feel.

## Where to save

Put your generated image(s) in this folder:

```
docs/style/
```

Suggested names (any of these work; more than one is fine):

- `moodboard.png` (or `.jpg`)
- `moodboard-01.png`, `moodboard-02.png`, ...
- `palette.png` for a dedicated color-swatch render

Once it's here, it will be reviewed and used to:

1. Re-define `CONFIG.colors` in `src/config.ts` (extract exact hex values).
2. Refine the style guidance + per-sprite prompts in `docs/Worn-Out-Sprites.md`.
